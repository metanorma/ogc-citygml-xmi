--

--

[.component,class=part]
--
实现规范**应**表示与 UML 类相同的源、目标、方向、角色和基数的关联。
--

[.component,class=part]
--
实现规范**应**表示 UML 类的属性，包括名称、定义、类型和基数。
--

[.component,class=part]
--
实现规范**应**表示所有父类的 UML 类属性，包括名称、定义、类型和基数。
--
