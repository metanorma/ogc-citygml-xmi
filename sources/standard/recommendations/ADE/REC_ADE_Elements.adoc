[[rec_ade_elements]]
[.recommendation]
====
[%metadata]
标签:: /rec/ade/elements
ADE 通常通过定义新的要素类型及附加内容，如对象类型、数据类型、代码列表和枚举，来扩展 CityGML。

[.component,class=part]
--
ADE **应该**使用 CityGML 中的预定义类型或 ISO 19100 系列国际标准的标准化架构。
--

[.component,class=part]
--
模型元素的约束**应该**使用正式语言表达，例如对象约束语言（OCL）。
--

[.component,class=part]
--
_&#171;core:AbstractSpace&#187;_ 的 ADE 子类**应该**包括约束，限制空间对象的边界。
--
====
