
[[req_ade_uml]]
[recommendation, ,type="general"]
====
[%metadata]
标签:: /rec/ade/uml

除了满足 CityGML ADE 的要求外，ADE 应该：

[.component,class=part]
--
CityGML 概念模型中使用的 UML 符号和构造（<<uml_notation_section>>）**应该**应用于 ADE 中相应的模型元素。
--

[.component,class=part]
--
ADE **应该**导入并使用来自外部概念 UML 模型的预定义类，例如 CityGML 模块或 ISO 19100 系列国际标准的标准化架构。
--
====
