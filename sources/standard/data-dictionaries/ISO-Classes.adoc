[[GM_Surface-section]]
==== GM_Surface（来自于 <<iso19107>>）

.GM_Surface（类）的元数据
[cols="1a,4a"]
|===
h|定义：|GM_Surface是GM_Primitive的一个子类，是二维几何的基础。不可定向的表面（如莫比乌斯带）是不允许的。曲面的方向通过选择向上法线来选择“上”的方向，如果曲面不是循环，则向上法线是曲面的一侧，外部边界从该侧逆时针显示。反转曲面方向将反转每个边界组件的曲线方向，并交换曲面的概念性“上”和“下”的方向。如果曲面是实体的边界，则“上”方向通常是向外。对于没有边界的闭合曲面，向上方向是曲面片的方向，曲面片必须彼此一致。其所包括的GM_SurfacePatches描述了GM_Surface的内部结构。

NOTE: 除了对可定向性的限制外，GM_Surface不需要其他“有效性”的限制条件。
