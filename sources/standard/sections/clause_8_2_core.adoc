CityGML 核心模块定义了城市模型的基本概念和组件。这项相当庞大的工作分为七个部分。这些部分相互依存，从相关 ISO 标准规定的基本原则逐步构建到完整的 CityGML 模型。这些部分总结在<<citygml-core-table>>.

[[citygml-core-table]]
.CityGML 核心部分
[cols="2.^,6.^"]
|===
| (<<ISO-dependencies-section>>)中 ISO 标准的使用 |描述了使用 ISO 19100 系列国际标准为 CityGML 模型提供基础。
|(<<city-objects-section>>)中城市模型和城市对象|定义了 CityGML 模型的基本构建块。
|(<<space-concepts-section>>)中空间概念|定义了 CityGML 模型中使用的空间概念。
|(<<geometry-lod-section>>)中几何和细节层次|定义了几何和细节层次（LOD）概念。
|(<<CityGML-section>>)中 CityGML 核心模型|展示了完整的核心模型。
|(<<b-e-c-section>>)中数据类型、枚举和代码列表 |定义了使该模型正常工作的一些细节内容。
|===
