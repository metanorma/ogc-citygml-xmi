外观模块以纹理和材质的形式提供表面数据的表示，例如表面几何对象的可观测属性。

外观不仅限于视觉数据，还代表称为主题的任意类别，如红外辐射、噪音污染或地震引起的结构应力。单个表面几何对象可能具有多个主题的表面数据。同样，表面数据可以由多个表面几何对象共享（例如道路铺设）。

在整个表面上恒定的表面数据根据 X3D 和 COLLADA 标准中的材质定义被建模为材质。依赖于表面内确切位置的表面数据被建模为纹理。这可以是参数化纹理（使用纹理坐标的纹理）或用于参数化的变换矩阵，或者是地理参考纹理（使用平面投影的纹理）。

每个表面几何对象在每个主题和每个面上都可以同时具有材质和纹理。这允许同时提供对表面属性的恒定近似值和复杂测量值。
