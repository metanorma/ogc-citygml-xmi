== 抽象测试套件
=== 引言

城市GML 3.0是一个概念模型。由于它与实现技术无关，因此无法提供可执行的测试脚本。提供符合性证据的责任落在实现规范上。此证据应作为附录包含在实现规范文档中。

本抽象测试套件中规定的测试方法是人工检查。在存在自动化方法的情况下，可以使用自动化方法。
