== 摘要

本标准定义了用于存储与交换虚拟三维城市模型的CityGML概念模型，由统一建模语言（Unified Modeling Language, UML）进行定义。
该UML模型在国际标准化组织211技术委员会（ISO/TC 211）的时空数据概念模型标准的基础之上构建，可确保城市模型中描述的人造要素与其所在的周围乡村共享统一的时空参考体系。

研制CityGML概念模型的核心目的是提供对三维城市模型中基本实体、属性和关系的统一定义，以允许在不同应用领域中重复利用相同的数据
，有助于三维城市模型的可持续、低成本维护。

本标准中描述的UML类模型亦可通过以下链接获取：
https://github.com/opengeospatial/CityGML3-Workspace/tree/1.0/UML/CityGML
