土地利用模块定义了可用于描述地球表面专用于特定土地利用或具有特定土地覆盖（有无植被均可，如沙地、岩石、泥滩、森林、草原或湿地，即物理外观）区域的对象。土地利用和土地覆盖是不同的概念。前者描述地球表面的人类活动，后者描述其物理和生物覆盖。然而，这两个概念相互关联，在实践中常常混淆。CityGML中的土地利用对象支持这两个概念：它们可用于表示地块、空间规划对象、休闲娱乐对象以及以3D形式描述区域物理要素的对象。在UML模型中，土地利用对象由顶级要素类型LandUse表示，这也是土地利用模块的唯一类。

土地利用模块的UML图如 <<landuse-uml>>所示。关于此需求类的详细讨论可以在 link:http://docs.opengeospatial.org/DRAFTS/20-066.html#ug-model-land-use-section[OGC CityGML 3.0 Users Guide]中找到。
