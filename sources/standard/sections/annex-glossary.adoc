== 术语表

=== 通用术语

==== 一致性测试类

必须应用的一组一致性测试模块，以获得单一的一致性证书

*来源*: xref:ogc08-131[OGC 08-131]，clause=4.4

==== 要素

对现实世界现象的抽象

*来源*: xref:iso19101-1[ISO 19101-1]，clause=4.1.11

==== 要素属性

要素的一个属性

*来源*: xref:iso19101-1[ISO 19101-1]，clause=4.1.12

==== 要素类型

具有共同特性的要素类别

*来源*: xref:iso19156[ISO 19156]，clause=4.7

==== 测量

以确定量值为目的的一组操作

*来源*: xref:iso19101-2[ISO 19101-2]，clause=3.21

==== 模型

对现实某些方面的抽象

*来源*: xref:iso19109[ISO 19109]，clause=4.15

==== 观测

测量或以其他方式确定属性值的行为

*来源*: xref:iso19156[ISO 19156]，clause=4.11

==== 观测程序

可用于进行观测的方法、算法或仪器，或这些的系统

*来源*: xref:iso19156[ISO 19156]，clause=4.12

==== 观测结果

通过已知观测程序确定的属性值的估计

*来源*: xref:iso19156[ISO 19156]，clause=4.14

==== 属性

通过名称引用的对象的一个方面或属性

*来源*: xref:iso19143[ISO 19143]，clause=4.21

==== 要求类

必须满足的所有要求模块的集合，以满足符合性测试类别

*来源*: xref:ogc08-131[OGC 08-131]，clause=4.19

==== 模式

模型的正式描述

*来源*: xref:iso19101-1[ISO 19101-1]，clause=4.1.34

==== 传感器

一种观测程序，它在其输出端提供观测属性的估计值

*来源*: xref:ogc08-094r1[OGC 08-094r1]，clause=4.5

==== 标准化目标

标准的某些要求适用的实体

*来源*: xref:ogc08-131[OGC 08-131]，clause=4.23

==== 时间序列

按时间顺序排列的数据值序列

*来源*: xref:ogc15-043r3[OGC 15-043r3]

==== 论域

包括所有感兴趣事物的真实或假设世界的视图

*来源*: xref:iso19101-1[ISO 19101-1]，clause=4.1.38

==== 版本

空间对象的特定变体

*来源*: xref:inspire_glossary[INSPIRE Glossary]

=== ISO概念

以下概念来自ISO TC211协调的UML模型，被城市GML概念UML模型引用，但在其定义中并不起主要作用。在此提供这些概念是为了支持对模型更完整的理解。

==== 面状

任何拓扑上为2D的几何对象的物理范围的度量。通常以长度的“平方”单位来测量。

*来源*: xref:iso19103[ISO 19103]

==== 布尔值

布尔值是与二值逻辑相关的数学数据类型

*来源*: xref:iso19103[ISO 19103]

==== CC_CoordinateOperation

对坐标进行数学运算，将坐标转换或变换到另一个坐标参考系统

*来源*: xref:iso19111[ISO 19111]

==== 字符

标准字符集中的一个符号

*来源*: xref:iso19103[ISO 19103]

==== 字符串

CharacterString是一系列标准字符集中的符号的数据类型家族

*来源*: xref:iso19103[ISO 19103]

==== CRS

坐标参考系统，通常是单一的，但也可以是复合的

*来源*: xref:iso19111[ISO 19111]

==== CV_DiscreteCoverage

CV_Coverage的一个子类，它为在其时空域内的单个几何对象中的任何直接位置返回单个记录的值

*来源*: xref:iso19123[ISO 19123]

==== CV_DomainObject

CV_Coverage的域的一个元素。它是一个对象的聚合，可以包括ISO 19107:2003中的任何GM_Objects组合、ISO 10108中的TM_GeometricPrimitives，或在其他标准中定义的空间或时间对象，例如在本国际标准中定义的CV_GridPoint

*来源*: xref:iso19123[ISO 19123]

==== CV_GridPointValuePair

CV_GeometryValuePair的一个子类型，其几何属性的值为GM_GridPoint

*来源*: xref:iso19123[ISO 19123]

==== CV_GridValuesMatrix

由各种偏移向量表示的几何在网格的图像平面内

*来源*: xref:iso19123[ISO 19123]

==== CV_ReferenceableGrid

CV_Coverage的一个子类，它将网格坐标与外部坐标参考系统联系起来

*来源*: xref:iso19123[ISO 19123]

==== 日期

日期给出年、月和日的值。日期的表示在ISO 8601中指定。日期和时间的原则在ISO 19108中进一步讨论

*来源*: xref:iso19103[ISO 19103]

==== 日期时间

日期时间是日期和时间类型的组合。日期时间的表示在ISO 8601中指定。日期和时间的原则在ISO 19108中进一步讨论

[.source]
<<iso19103>>

==== 距离

用于返回距离和可能的长度

*来源*: xref:iso19103[ISO 19103]

==== 工程CRS

一种上下文局部坐标参考系统，可分为两大类：

. 应用于地球表面或附近的工程活动的地球固定系统；

. 在移动平台上使用的CRS，如道路车辆、船只、飞机或航天器。

*来源*: xref:iso19111[ISO 19111]

==== 要素类型

作为代表各个要素类型的类的实例化的元类

*来源*: xref:iso19109[ISO 19109]，clause=7.4.5

==== 通用名称

通用名称是命名空间中所有名称的抽象类。通用名称的每个实例要么是本地名称，要么是作用域名称

*来源*: xref:ogc08-131[OGC 08-131]，clause=4.18

==== 几何

几何是地理参考几何对象分类的根类，并支持所有地理参考几何对象共有的接口

*来源*: xref:iso19107[ISO 19107]


==== GM_CompositePoint

包含一个且仅一个GM_Point的GM_Complex

*来源*: xref:iso19107[ISO 19107]

==== GM_CompositeSolid

一组几何实体，它们沿着公共边界几何表面相互邻接

*来源*: xref:iso19107[ISO 19107]

==== GM_GenericSurface

GM_Surface和GM_SurfacePatch都表示表面几何的一部分，因此共享许多操作签名。这些在接口类GM_GenericSurface中定义

*来源*: xref:iso19107[ISO 19107]


==== GM_LineString

由一系列线段组成，每个线段都有类似于GM_LineSegment的参数化

*来源*: xref:iso19107[ISO 19107]


==== GM_MultiPrimitive

所有原始集合的根类。关联角色“element”应是包含在此GM_MultiPrimitive中的GM_Primitives集合。此处的属性声明专门化了GM_Aggregate中的声明，仅包括此类型集合中的GM_Primitives

*来源*: xref:iso19107[ISO 19107]


==== GM_OrientableSurface

一个表面和从GM_OrientablePrimitive继承的方向。如果方向是“+”，则GM_OrientableSurface是一个GM_Surface。如果方向是“-”，则GM_OrientableSurface是对一个GM_Surface的引用，该GM_Surface的upNormal反转了此GM_OrientableSurface的方向，即“表面顶部”的方向

*来源*: xref:iso19107[ISO 19107]


==== GM_PolyhedralSurface

由沿公共边界曲线连接的多边形表面（GM_Polygon）组成的GM_Surface

*来源*: xref:iso19107[ISO 19107]


==== GM_Position

由直接位置或对GM_Point的引用组成的联合类型，应从该GM_Point获取直接位置

*来源*: xref:iso19107[ISO 19107]


==== GM_Primitive

几何原始的抽象根类。其主要目的是定义基本的“边界”操作，将每个维度中的原始联系在一起

*来源*: xref:iso19107[ISO 19107]


==== 整数

精确的整数值，没有小数部分

*来源*: xref:iso19103[ISO 19103]


==== 物联网

物联网是物理对象——“事物”——的网络，这些对象嵌入了传感器、软件和其他技术，用于通过互联网与其他设备和系统连接和交换数据。+

*来源*: xref:iso19103[ISO 19103]

==== IO_IdentifiedObjectBase

CRS或与CRS相关对象的补充标识和注释信息

*来源*: xref:iso19111[ISO 19111]


==== 长度

作为积分的测量距离，即曲线上点之间的距离的无限和的极限

*来源*: xref:iso19107[ISO 19107]


==== 测量

执行确定实体范围、尺寸或数量的行为或过程的结果

*来源*: xref:iso19103[ISO 19103]

==== 数字

所有数字数据的基类型，提供基本的代数运算

*来源*: xref:iso19103[ISO 19103]

==== 点

GM_Point是仅由一个点组成的几何对象的基本数据类型

*来源*: xref:iso19107[ISO 19107]

==== 实数

使用基数2的常见二进制实数有限实现

*来源*: xref:iso19103[ISO 19103]

==== RS_ReferenceSystem

数据集使用的空间和时间参考系统的描述

*来源*: xref:iso19111[ISO 19111]

==== 作用域名称

作用域名称是由定位另一个命名空间的本地名称和在该命名空间中有效的通用名称组成的复合体。作用域名称包含一个本地名称作为头部和一个通用名称（可以是本地名称或作用域名称）作为尾部

*来源*: xref:iso19103[ISO 19103]

==== 实体

GM_Solid是GM_Primitive的一个子类，是三维几何的基础。实体的范围由边界表面定义

*来源*: xref:iso19107[ISO 19107]

==== 时间

时间是天文或原子时间尺度上某一瞬间的名称。它用于表示一天中的时间

*来源*: xref:iso19103[ISO 19103]

==== TM_Duration

用于描述时间维度中的长度或距离的数据类型

*来源*: xref:iso19108[ISO 19108]

==== TM_TemporalPosition

TM_Instant相对于TM_ReferenceSystem的位置

*来源*: xref:iso19108[ISO 19108]

==== 测量单位

为测量距离或面积等物理量而设计的任何系统，或为测量时间流逝等事物而设计的系统

*来源*: xref:iso19103[ISO 19103]

==== URI

统一资源标识符（URI）是一个紧凑的字符字符串，用于标识或命名资源

*来源*: xref:iso19103[ISO 19103]

==== 体积

任何三维几何对象的物理空间的度量

*来源*: xref:iso19103[ISO 19103]
