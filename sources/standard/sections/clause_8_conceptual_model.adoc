== CityGML UML 模型

CityGML UML 模型是 CityGML 概念模型的规范性定义。本节中的表格和图形由软件根据 UML 模型生成。因此，本节提供了 CityGML 概念模型的规范性表示。

另一种表示形式可在<<data-dictionary-section>>中找到。

[[rc_overview-section]]
=== 需求类的结构概述

本标准的需求类被构建为 UML 包，如图 <<package-diagram>>所示。每个需求类在各自的子部分中都有详细说明。这些子部分包括 UML 图、数据字典和适用的需求。
