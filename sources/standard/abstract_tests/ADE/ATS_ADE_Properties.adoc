[.component,class=测试目的]
--
验证对CityGML概念模型的应用领域扩展（ADE）是否正确实现了扩展属性。
--

[.component,class=测试方法类型]
--
人工检查
--

[.component,class=测试方法]
=====

[.component,class=step]
======
CityGML概念模型中的每个要素类型都包含一个扩展属性，其目的是允许ADE向该现有的要素类型添加属性。在每个使用了扩展属性的情况下：

[.component,class=step]
--
验证ADE是否创建了与扩展属性相关联的抽象数据类型的子类。
--

[.component,class=step]
--
验证此子类是否被定义为使用_«DataType»_构造型的数据类型。
--

[.component,class=step]
--
验证该要素类型的所有特定于应用程序的属性和关联是否被建模为ADE子类的属性。
--
