[abstract_test]
====
[%metadata]
标签:: /ats/ade/elements
主体:: <<req_ade_elements,/req/ade/elements>>

[.component,class=测试目的]
--
验证对CityGML概念模型的应用领域扩展（ADE）是否正确实现。
--

[.component,class=测试方法类型]
--
人工检查
--

[.component,class=测试方法]
=====

[.component,class=step]
======
对于每个由ADE定义的新UML类：

[.component,class=step]
--
验证ADE中的每个要素类型类是否直接或间接继承自CityGML根要素类型_core:AbstractFeature_或其子类。
--

[.component,class=step]
--
验证ADE中代表顶级要素类型的每个UML类是否被赋予了_«TopLevelFeatureType»_构造型。
--

[.component,class=step]
--
验证ADE中代表空间或空间边界的每个UML类是否分别直接或间接继承自_core:AbstractSpace_或_core:AbstractSpaceBoundary_。
--

[.component,class=step]
--
验证ADE定义的任何新的或额外的空间属性：

. 属于预定义的LOD，
. 属性名称以“lod__X__”为前缀，其中_X_是一个表示目标LOD的整数值。
--
