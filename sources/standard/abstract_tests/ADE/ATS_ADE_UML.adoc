[abstract_test]
====
[%metadata]
标签:: /ats/ade/uml
主体:: <<req_ade_uml,/req/ade/uml>>
[.component,class=测试目的]
--
验证对CityGML概念模型的应用领域扩展（ADE）是否正确地以UML形式建模。
--

[.component,class=测试方法类型]
--
人工检查
--

[.component,class=测试方法]
=====

[.component,class=step]
======
ADE按照ISO 19100系列国际标准的概念建模框架以UML定义为概念模型

[.component,class=step]
--
验证ADE UML模型是否符合ISO 19109中规定的通用要素模型。
--

[.component,class=step]
--
验证ADE UML模型是否符合ISO/TS 19103中规定的应用模式的规则和约束。
--

[.component,class=step]
--
验证ADE UML模型是否被组织成一个或多个具有全球唯一命名空间的UML包，并包含ADE定义的所有UML模型元素。
--
